函数

函数式编程

在大多数简单的术语中，函数式是一种编程形式，你可以将函数作为参数传递给其他函数，并将函数作为值返回。 在函数式编程中，我们以函数的形式思考和编程。

JavaScript，Haskell，Clojure，Scala 和 Erlang 是部分实现了函数式编程的语言。

一等函数

　

JavaScript 将函数视为一等公民。 那是因为在 JavaScript 及其他函数式编程语言中，函数是对象。

在 JavaScript 中，函数是一种特殊类型的对象。 它们是 Function objects。

在 JavaScript, 可以像操作其他对象一样操作函数。

高阶函数

高阶函数是对其他函数进行操作的函数，简单来说，高阶函数就是接收一个函数作为参数的函数，或者是把函数作为返回值的函数。

JavaScript 中的内置函数有很多都是高阶函数，比如操作数组的很多函数，map()、filter()、some() 等等。

命名空间

命名空间本质上就是一个普通的对象，将相关的变量和函数封装起来，做为该对象属性和方法。

作用：解决变量污染的问题。

自执行函数

自执行函数是将函数的声明和调用合并在一起。

作用：

  • 封装，通过命名空间或自执行函数实现程序的基本封装，提升程序的可维护性。

  • 两种写法

  • 自执行函数属于高阶函数

闭包

一个函数的返回值是另一个函数。

闭包是一个携带内部参数的函数，并且这个参数对外是完全隐藏的，不会受到外面的影响。
