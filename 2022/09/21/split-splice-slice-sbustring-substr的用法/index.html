split,splice,slice,sbustring,substr的用法

2022-09-21

split、splice、slice、substring、substr 这几个方法是不是经常看到，是不是经常迷惑， 是不是经常有种【好像是这个，但又不确定】的时候，这篇文章就把这几个方法都整理出来，再也不怕搞混了。

split(char) String

使分裂（成不同的派别）；分开；分担；分摊

  • char：切分的字符

用来切分字符串，以某一个字符切分成数组，返回切分之后的数组，数组中不包括切分的字符。

splice(index,howmany,item1…itemx) Array

拼接；接合

  • Index：开始的数组索引，为负数时，从尾部开始，不传为 0

  • howmany：删除几个元素

  • item1：添加的第一个元素

  • itemx：添加的第x个元素

用来删除数组中的项，返回被删除的项，会改变原数组。

slice(start,end) 左开右闭 Array 、String

切成片；削球；被切成片

用来从一个数组当中切取部分元素，返回一个新的数组，不会改变原数组。

  • 两个参数可以是负数，为负数时，参数与长度相加，也就是从后往前数

substring(start,end) 左开右闭 String

用来从一个字符串当中截取部分数据，返回一个新的字符串。

  • 参数为负数时，自动转换为 0

substr(start,length) String 不推荐使用，标准里没有规定

用来从一个字符串当中截取部分数据，返回一个新的字符串。

  • start：开始位置，为负数时与长度相加， 也就是从后往前数

  • length：要截取的长度，为负数时自动转换为 0
