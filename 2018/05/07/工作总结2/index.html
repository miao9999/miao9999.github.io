2018 工作总结

2018-05-07

工作总结

最近连续加班快有一个月了，这是第一次时间最长，强度最大的一次加班，另外来这里工作也快半年了，总结一下吧。

  1. 心态。
    以前工作中，遇到问题会特别紧张，尤其是从未见过的问题，更加紧张，在问题没有解决之前用寝食难安来形容一点都不夸张，上家公司的老板开会时经常会说，遇到问题不要着急，因为肯定会解决的，当时完全做不到，不管用哪种方式，都没有办法缓解这种面对问题时的压力，所以这也成了我最担心的问题之一。而这几天下来，发现遇到问题时不会再那么紧张了，甚至还会从容一些，终于体会到了前老板说的『问题总会解决的』。这一点可以说是这段时间最大的收获。
  2. 技术
    从技术上来说，这段时间的提升并没有太多，一直都在赶着做业务上的功能，没有在技术层面做更多深入的研究。反而是接触的范围比较广泛，很多技术是之前没有接触过的，在这段时间也有了一些基本的了解，在技术方面的提升大概就是这些了。
  3. 态度
    不管是学习还是工作，能否取得好的成绩，态度是很重要的一个因素。但在对待学习和工作时的态度还是有很大的差距。在工作一定会有很多不合自己心意的任务，这时在对待这个任务的态度该是如何的呢？目前多见过的面对这样的情况的态度也有好几种了，有的默默接受，有的很不满，但最终也会接受，还有的表面接受，但其实并不会认真对待，还有很多，目前我遇到的就是这几种。
    这个时候应该很多人都会想，最好的态度就是去沟通，要把自己的想法表达出来，要让工作更加合理才对。从书上找的话，找到这个答案的几率应该也会很高。但是这几种到底哪个对，我不知道。很多人认为的最好的太度，我目前还没见过身边有哪个人做到的，大多数都是前面几种，但是不同的态度有时候的结果却并不会像你想你中的那样，所以我不知道哪个是对的，也不知道我在面对这样的问题的时候我应该用哪种态度去面对。
    在这方面还有待提升。
  4. 与其他人的合作
    在这个方面，目前还没有发现有什么太大的毛病，或是收获。自己没发现问题，但不意味着就没有问题，也许有很问题，只是这些问题自己没发现，而对方也不好意思讲。其实我还是希望对方能讲出来，虽然听的时候心里真的会有不开心，但还是很感谢他的。有些问题真的是自己发现不了，尤其是一些由于固有思维导致的问题，可能自己从来都没觉得有什么不妥，也从来没想过这回事，但它就是存在问题的，这种问题只有别人才能发现。所以我很期待会有人指出我的不足，我会虚心接受，并努力改正。

以上这四点就是最近这段时间想的最多的，有些问题还没有解决或是没有彻底解决，做为下一个阶段的方向和目标吧！
