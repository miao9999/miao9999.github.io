1.1 初试 Kotlin

首先，Kotlin 到底是什么？它是一个针对 Java 平台的新的程序语言。Kotlin是简洁、安全、务实、注重与 java 代码的交互操作。Kotlin 几乎可以使用在任何使用 Java 的地方，如：服务器端的开发，Android apps 开发，或是其他的地方。Kotlin 可以很好运行 Java 现有的所有的库和框架，并且有和 Java 相同的性能水平。接下来让我们用一个简单的例子来展示 Kotlin 是什么样子的。这个例子定义一个 Person 类，创建一个人的集合，找到其中年龄最大的那个，并把结果打印出来。虽然只有少量的代码，但你能看到很多 Kotlin 有趣的特点。我们已经在本书中标记了其中的一些，这样在以后的阅读中可以很容易地找到它们。代码作了一些简单的说明，如果有什么不清楚的地方现在也不用担心。在后面我们会详细的介绍。

如果你想要运行这个盒子，最简单的选择就是使用在线编辑器 https://try.kotl.in/ 在示例中输入代码，并点击 Run 按钮，代码就会被执行。下面是代码：

1.2 Kotlin 的主要特征

你可能已经知道 Kotlin 是什么样的语言。接下来让我们更详细的看看它的主要属性。首先，我们了解一下用 Kotlin 可以构建哪些应用。

1.2.1 目标平台：服务器端，Android，Java 运行的任何地方

Kotlin 的主要目标就是提供一个更简洁，更高效，更安全的替代品，适用于现在所有 Java 使用的场合。Java 是一种非常流行的语言，并且它被使用的环境的种类也是非常广泛，从智能卡（Java Card 技术）到 Google、Twitter、LinkdIn、其互联网规模的公司运行的最大的数据中心。在大多数的这些地方，使用 Kotlin 可以帮助开发者用更少的代码，更少的烦恼从而实现自己的目标。

以下是 Kotlin 最常用的领域：

  • 构建服务器端代码（典型的，web 应用的后端）
  • 构建运行在 Android 设备上的移动应用

但是 Kotlin 也可以运行在其他的地方。例如，你可以使用 Intel MultiOS Engine ^1^ 在 iOS 设备上运行 Kotlin 代码。你也可以同时使用 Kotlin 与 JavaFX ^2^ 构建桌面应用。

Footnote 1 https://software.intel.com/en-us/multi-os-engine

Footnote 2 Oracle，http://mng.bz/500y

除了 Java，Kotlin 也能编译 JavaScript，允许你运行 Kotlin 代码在浏览器。但是在撰写此文时，JavaScript 的支持仍处在探索并且原型在 JetBrains，所以这个超出了本书的范围。其他平台也在考虑未来语言的版本。

正如你看到的，Kotlin 的目标是非常广泛的。Kotlin 并没有专注于软件开发人员今天面临的单一的问题领域或地址单一型的挑战。相反的，它为开发过程中出现的所有任务提供全面的生产力改进。它提供了与支持特定域或编程范例的库的良好集成级别。让我们看一下 Kotlin 作为编程语言的主要作用。
