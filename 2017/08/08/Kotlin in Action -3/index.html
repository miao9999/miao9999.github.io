1.2.4 免费开源

Kotlin 语言，包括编译器、库、所有相关的工具都是完全开源的，并且免费用于任何目的。它在Apache 2许可证的获得，开放在 GitHub ^3^ 上，欢迎在为社区贡献。可以选择三个开源 IDE 开发 Kotlin 应用：IntelliJ IDEA Community Edition、Android Studio、Eclipse 这三个都是支持的。（IntelliJ IDEA Ultimate也可以开发）

角标 3：http://github.com/jetbrains/kotlin

现在可以理解 Kotlin 是一种什么样的语言，接下来看一下在实际的应用中它的优势。

1.3 Kotlin 的应用

正如我们之前提到的，Kotlin 主要使用在服务端和 Android 端开发两个领域。下面依次讲解这两个领域，并看看 Kotlin 为什么适合它们。

1.3.1 Kotlin 用于服务端

服务端编程是一个非常宽泛的概念。它包含了以下几种以及更多的应用的类型：

  • 网页应用返回一个 HTML 页面到浏览器
  • 移动应用后端是一个在 HTTP 上暴露出一个 JSON API
  • 在 RPC 协议上，与其他微服务通讯的微服务

开发者们用 Java 已经构建了这几种应用，并且已经累积了很多的框架和技术帮助构建它们。这种应用通常不是孤立开发的，也不是从零开始的。几乎总是有一个现有的系统正在被扩展、改进或替换，新的代码必须与系统的现有部分集成，这可能是多年前编写的。Kotlin 在这种环境中最大的优势是它可以和 Java 代码无缝结合。不管是正在写一个新的组件还是把现有的服务端的代码迁移到 Kotlin 中，Kotlin 都会很好的适应。当你在 Kotlin 中继承 Java 类时，或是以某种方式注释类中的方法或字段时都不会遇到问题。这样的好处是系统的代码会更简洁，更可靠，更容易维护。

同时，Kotlin启用了大量的新技术，开发这样的系统。例如，它支持生成器模式，允许您创建具有简明语法的任何对象图，在语言中保留完整的抽象集合和代码重用工具。
