mac 手动安装 gradle

2017-05-23

在 gradle 的官网里提供了两种安装方式，一种是用 brew 安装，一种是手动安装。用 brew 安装实在是太慢了，所以我选择用 gradle 安装。安装并不难，重要的是路径的配置，坑了好多次了，这个坑必须填一下了。

  1. 下载 gradle

  2. 解压文件

  3. 配置路径

    1. 终端配置，只能在当前会话内使用，关闭后，再打开，需要重新配置

      export PATH=$PATH:gradle 的路径（如果相对路径不行，就设置绝对路径，把 gradle 的文件夹拖到终端就可以看到了）

    2. 在 .bash_profile 文件中配置，这种方式只配置一次，之后都不用再配置

      1. 在终端中输入 open -e .bash_profile ,打开 .bash_profile 文件
      2. 在 .bash_profile 文件中输入export PATH=$PATH:gradle 的路径/bin ，保存文件
      3. 在终端中执行 source ~/.bash_profile,更新.bash_profile 文件
      4. 到此路径配置完成，在终端中输入 gradle -v 就可以查看 gradle 的版本信息，但现在如果关闭终端，再次打开，路径就失效了。所以还需要以下的设置。
      5. 在终端输入 vim ~/.zshrc ，之后在.zshrc 文件中加入一句 source ~/.bash_profile 保存并退出。这样每次打开终端都不用再重新配置路径了。
